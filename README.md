## Description

Soft Market is a groccery online store built in MVC approach using the following tech stack:
- [Nest](https://github.com/nestjs/nest) (a JavaScript framework for building and efficient and scalable Node.js server-side applications that fully support TypeScript)
- [Bootstrap](https://getbootstrap.com/) (a popular CSS framework)
- [Handlebars](https://handlebarsjs.com/) (a simple templating language)
- [MySQL](https://www.mysql.com/) (a popular Open-Source SQL database management system)
- [TypeORM](https://typeorm.io/) (an ORM that can run in NodeJS and can be used with TypeScript and JavaScript)


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
